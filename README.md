# Usage

1. Download the project

```bash
$ git clone https://github.com/JavelFreitas/NLW-Rocketseat.git 
```

2. NodeJS

```bash
$ cd ./server
$ npm install
$ npm run knex:migrate
$ npm run knex:seed
$ npm run dev
```

3. React

```bash
$ cd ./web
$ npm install
$ npm start
```
